# Blockchain dengan framework Hyperledger Fabric v2.5 untuk Linux

1. Download assets dari link HLF berikut 
```
https://github.com/hyperledger/fabric/releases?q=2.5&expanded=true
```

2. Pilih salah satu yang sesuai, contoh disini menggunakan
```
hyperledger-fabric-linux-amd64-2.5.14.tar.gz
```

3. Kemudian extract filenya
```
tar -xvzf hyperledger-fabric-linux-amd64-2.5.14.tar.gz
```
